### Loss Functions

* **BCE Loss**: 
  $$
  \begin{equation}
  J(\theta) = -\frac{1}{m} \sum_{i=1}^m [y^{(i)}log(\hat{y}^{(i)}) + (1 - y^{(i)})log(1 - \hat{y}^{(i)}) ]
  \end{equation}
  $$
